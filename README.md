## Instructions
### Requirements
#### -Node.js https://nodejs.org/en/download/
#### -SQLite3 https://www.sqlite.org/index.html
### Installing the modules
In your terminal, move to directory of the server then do 
#### "npm install" 
### Starting the Programme
In your terminal, move to directory of the server then do
#### "node server"
